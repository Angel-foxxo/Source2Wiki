Mesh Entity

Markup group entity; add the group's targetname to the 'Markup Groups' field of a markup entity to add that entity to this group

<details open>
<summary><h2>Keyvalues</h2></summary>
- **Markup Groups** (`groupnames`) \<`TagListDynamic`\>

- **Auto Group By Volume** (`groupbyvolume`) \<`Boolean`\>\
Automatically add any markup entities to this group that overlap this volume

- **Auto Group Other Groups** (`groupothergroups`) \<`Boolean`\>\
When auto-grouping (by prefab or by volume), include other markup_group entities in this group. By default this does not happen.


 </details>
