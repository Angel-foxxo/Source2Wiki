<summary><h2>Keyvalues</h2></summary>
- **Start Disabled** (`StartDisabled`) \<`Boolean`\>

- **Entity Scripts** (`vscripts`) \<`ScriptList`\>\
Name(s) of script files that are executed after all entities have spawned.

- **Name** (`targetname`) \<`TargetSource`\>\
The name that other entities refer to this entity by.

- **Cubemap Source** (`cubemapfogsource`) \<`Choices`\>\
Which field to use to select the cubemap and its parameters.
  - Cubemap From Env_Sky 
  - Cubemap From Material 


- **Env_Sky Target** (`cubemapfogskyentity`) \<`TargetDestination`\>

- **Sky Material** (`cubemapfogskymaterial`) \<`Resource`\>

- **Cubemap Blur / Mip Bias (0-1)** (`cubemapfoglodbiase`) \<`Float`\>\
Adjust how blurry the cubemap should be. Lower values are more blurry (0-1) 

- **Fog Start Distance** (`cubemapfogstartdistance`) \<`Float`\>

- **Fog End Distance** (`cubemapfogenddistance`) \<`Float`\>

- **Distance Falloff Exponent** (`cubemapfogfalloffexponent`) \<`Float`\>\
Exponent for distance falloff. For example, 2.0 = proportional to square of distance

- **Maximum Fog Opacity (0-1)** (`cubemapfogmaxopacity`) \<`Float`\>\
Set the maximum opacity of the cubemap fog. (0-1)

- **Height Fog Enabled?** (`cubemapheightfog`) \<`Boolean`\>

- **Height Fog Start** (`cubemapfogheightstart`) \<`Float`\>

- **Height Fog End** (`cubemapfogheightend`) \<`Float`\>

- **Height Fog Exponent** (`cubemapfogheightexponent`) \<`Float`\>\
Exponent for distance falloff. For example, 2.0 = proportional to square of distance

- Disable \<`Void`\>\

- RunScriptFile \<`String`\>\

- RunScriptCode \<`String`\>\
Execute a fragment of script code

- CallScriptFunction \<`String`\>\

- CallPrivateScriptFunction \<`String`\>\
Calls a script function from this entity's private script scope.

- CallGlobalScriptFunction \<`String`\>\
Calls a script function in the global script scope.

- Kill \<`Void`\>\
Removes this entity from the world.

- KillHierarchy \<`Void`\>\
Removes this entity and all its children from the world.

- KillConstrained \<`Void`\>\
Removes this entity, all its children, and anything constrained to it, from the world.

- AddOutput \<`String`\>\
Adds an entity I/O connection to this entity. Parameter format: outputname&gt;targetname>inputname&gt;parameter&gt;delay&gt;max times to fire (-1 == infinite). Very dangerous, use with care.

Causes this entity's OnUser1 output to be fired.

- FireUser2 \<`Void`\>\

- FireUser3 \<`Void`\>\
Causes this entity's OnUser3 output to be fired.

- FireUser4 \<`Void`\>\
