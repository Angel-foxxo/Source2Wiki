<summary><h2>Keyvalues</h2></summary>
- **Search Type** (`SearchType`) \<`Choices`\>\
How to search for the entities to change.
  - Entity Name 
  - Classname 
  - Old Hint Group 


- **Name to search for** (`SearchName`) \<`TargetNameOrClass`\>

- **New Hint Group** (`NewHintGroup`) \<`String`\>

- **Search Radius** (`Radius`) \<`String`\>\
Radius to search (0 for all of map)

- **Hint Limit Nav** (`hintlimiting`) \<`Boolean`\>\
Limits NPC to using specified hint group for navigation requests, does not limit local navigation.
