Mesh Entity

Tagged markup volume entity with a target reference point.  Points are only in volume if behind the reference point relative to a given query position.

<details open>
<summary><h2>Keyvalues</h2></summary>
- **Markup Groups** (`groupnames`) \<`TagListDynamic`\>

- **Ref Pos** (`ref_position`) \<`LocalPoint`\>\
Reference position.

- **Use Reference Position** (`use_ref_position`) \<`Boolean`\>\
Use the reference position when calculating validity of query positions.
